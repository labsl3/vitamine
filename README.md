# Vitamine

Basic virtual machine that aims to implement hot-code swapping primitives *a la Erlang*.
Project realized within the framework of the SUPINFO Low-Level Laboratory - SL3.

## Usage : 

```
mkdir bin && cd bin
cmake ..
./src/vitamine ../test/basic-arithmetic.s
```
